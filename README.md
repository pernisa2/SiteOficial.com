## 📝 Sobre o Projeto

Este site foi desenvolvido para servir como minha plataforma oficial de apresentação. Aqui, consolido minha jornada como estudante de Ciência da Computação e entusiasta de tecnologias como Python, PHP e cibersegurança. O objetivo é demonstrar não apenas o código, mas a visão por trás dos projetos que desenvolvo.

---

## 🚀 O que você encontra aqui

* **Portfólio de Projetos:** Uma curadoria dos meus melhores trabalhos, desde aplicações web em PHP até ferramentas de produtividade.
* **Jornada Acadêmica:** Um resumo do meu desenvolvimento acadêmico e experiências práticas na área de tecnologia e segurança da informação.
* **Contato Direto:** Canais facilitados para networking e oportunidades profissionais.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído priorizando performance e boas práticas de desenvolvimento Front-End:
* **HTML5 Semântico:** Estrutura clara e otimizada para navegadores.
* **CSS3 Moderno:** Design responsivo e adaptado para diferentes resoluções.
* **JavaScript:** Interatividade refinada para oferecer uma experiência de usuário fluida.
